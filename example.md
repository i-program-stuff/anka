# Anki Vocabulary JSON Format

Each chapter's vocabulary is stored as a JSON file containing an array of card objects. One file per chapter (e.g. `Chapter 1 Vokabel.json`).

## Card object fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `front` | string | yes | English translation / the side shown first |
| `back` | string | yes | German word/phrase. Nouns use the form `article Wort, plural-suffix` (e.g. `die Einrichtung, -en`). Verbs include case/complement markers (e.g. `jdm. etw. anvertrauen`, `sich einsetzen für (+ Akk.)`) |
| `note_id` | integer | yes | Unique numeric ID (epoch-milliseconds based, sequential). Unique across all chapters. |
| `type` | string | yes | One of: `noun`, `verb`, `adjective`, `adverb`, `phrase` |
| `gender` | string/null | nouns only | `der`, `die`, `das`, `der/die` (for gender-inclusive nouns like *der/die Streetworker/in*). `null` for non-nouns. |
| `plural` | string/null | nouns only | Dictionary-style suffix: `-e`, `-en`, `-n`, `-¨e` (umlaut), `-` (no change), `kein Pl.` (uncountable/abstract), `nur Pl.` (plural-only). `-en/innen` for gender-inclusive nouns (masc. pl. / fem. pl.). `null` for non-nouns. |
| `chapter` | integer | yes | Chapter number (1, 2, or 3) |
| `tags` | array of strings | yes | Typically `["chapter-N", "<type>"]`, e.g. `["chapter-1", "noun"]` |
| `notes` | string | yes | Grammar hints, clarifications, fix-log. Examples: `"separable"`, `"reflexive"`, `"+ Dat."`, `"irregular"`, `"idiom"`, `"original file translation corrected"`. Empty string `""` if nothing to note. |
| `example_de` | string | yes | Example sentence in German (B2 level) |
| `example_en` | string | yes | English translation of the example sentence |
| `perfekt` | string | verbs only | Perfekt form with auxiliary. Reflexive verbs include `sich`. Examples: `"hat überlebt"`, `"ist erfroren"`, `"hat sich regeneriert"`, `"hat/ist zerbrochen"` |
| `prateritum` | string | verbs only | Präteritum (3rd person singular). Separable prefixes split; reflexive `sich` included where appropriate. Examples: `"überlebte"`, `"erfror"`, `"wuchs heran"`, `"regenerierte sich"` |

## Type classification rules

- **noun**: Capitalised German word with an article (der/die/das) or clearly a noun (e.g. *das Weißblech*, *die Kündigungsfrist*). Includes nominalised expressions (*das Hier und Jetzt*, *das Muss*).
- **verb**: Infinitive, possibly with reflexive `sich`, object markers (`jdn.`, `jdm.`, `etw.`), or prepositional complements (`auf + Akk.`, `für + Akk.`). Single-word verbs or separable verb constructions are both `verb`.
- **adjective**: Descriptive words (*massiv*, *wasserdicht*, *sorgfältig*). Some also function as adverbs (*zeitnah*, *umgehend*) — classified as `adjective` with a note.
- **adverb**: Exclusively adverbial words (*verhältnismäßig*, *idealerweise*, *bekanntlich*).
- **phrase**: Fixed expressions, idioms, and multi-word constructions that are not single verbs (*ein offenes Ohr haben*, *im Hinblick auf*, *in öffentlicher Hand*, *einen wahren Kern haben*).

## Plural notation (nouns)

| Suffix | Meaning | Example back |
|--------|---------|-------------|
| `-e` | add -e | `das Gefäß, -e` |
| `-en` | add -en | `die Einrichtung, -en` |
| `-n` | add -n | `die Anekdote, -n` |
| `-` | no change | `der Behälter, -` |
| `¨-e` | umlaut + -e | `der Schicksalsschlag, ¨-e` → Schicksalsschläge |
| `¨-e` | umlaut + -e | `der Umschlag, ¨-e` → Umschläge |
| `kein Pl.` | no plural (abstract/uncountable) | `die Geborgenheit, kein Pl.` |
| `nur Pl.` | plural-only noun | `die Regularien (nur Pl.)` |
| `-/innen` | gender-inclusive masc./fem. plural | `der/die Streetworker/in, -/innen` |

## note_id generation

Generated by Python command at build time:
```python
base = int(time.time() * 1000)  # epoch milliseconds
note_ids = [base + i for i in range(total_cards)]
```
These are written directly into the JSON — do not regenerate on import.
